# Decision Journal

> Append-only log of all significant project decisions.
> Each entry documents the context, options considered, decision made, and rationale.

---

```markdown

**Skill**: skill-name
**Complexity**: Low | Medium | High
**Phase**: requirements | research | architecture | implementation | integration | qa | deployment | continuous

### Context
[What situation or problem prompted this decision?]

### Options Considered
1. **Option A**: [Description]
   - Pros: [...]
   - Cons: [...]

2. **Option B**: [Description]
   - Pros: [...]
   - Cons: [...]

### Decision
[What was decided?]

### Rationale
[Why was this option chosen over others?]

### Tradeoffs
- (+) [Positive tradeoff]
- (-) [Negative tradeoff]

### Reversibility
[High | Medium | Low] - [What would it take to reverse this decision?]

```
