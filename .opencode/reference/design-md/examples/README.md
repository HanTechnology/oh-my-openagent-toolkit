## How To Use These Snapshots

- Start with the catalog to pick one primary reference.
- Add one secondary reference only when it clarifies a separate trait, such as density, motion, tone, or commerce polish.
- Extract reusable patterns like hierarchy, contrast, spacing, tokens, copy posture, and interaction rhythm.
- Don't copy logos, proprietary art, exact copy, or a third-party brand system.
